# Parking Management System (C)

A console-based Parking Management System written in C.

## Features
- Admin login system
- Separate parking for 2-wheelers and 4-wheelers
- First-fit slot allocation
- System time-based entry and exit tracking
- Minimum 1-hour billing with rounding
- Automatic bill generation at exit
- Text file handling for data persistence
- Vehicle search functionality

## How to Compile and Run
```bash
gcc parking_c.c -o parking
